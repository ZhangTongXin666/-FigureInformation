# 一、目标人群 <br>
我们这款软件以专业期刊杂志信息的提供为核心，便捷和多功能性为辅，主要目标群体为专业领域人士和某些专业知识的爱好者，为他们提供他们感兴趣领域的最新消息，免去他们购买期刊和翻阅期刊的烦恼。<br><br>
